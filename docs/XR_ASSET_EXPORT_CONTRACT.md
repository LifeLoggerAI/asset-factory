# XR Asset Export Contract

Asset Factory exports deterministic and provider-backed outputs for URAI consumers.

## Required export formats
- PNG
- SVG
- WebP
- GLB
- GLTF
- texture/material metadata JSON

## Consumer repos
- UrAi
- urai-spatial
- urai-studio
- B2Bportal demos

## Required metadata
- `id`
- `assetType`
- `format`
- `uri`
- `status`
