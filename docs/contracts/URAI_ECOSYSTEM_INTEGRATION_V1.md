# URAI Ecosystem Integration Contract V1

Asset Factory is the generated-asset owner in the URAI chain:
`UrAi -> urai-jobs -> urai-content -> asset-factory -> urai-spatial -> urai-studio -> B2Bportal`

Use `URAI_ECOSYSTEM_SCHEMA_V1.json` for data compatibility.

Asset-specific obligations:
- Produce deterministic assets and metadata in supported formats (`png`, `svg`, `webp`, `glb`, `gltf`).
- Preserve sample packs for orb, ground, sky, emotional field, memory bloom, LifeMap node, spatial environment, AR anchor, and VR object.
- Publish clear failure reasons when generation cannot proceed.
