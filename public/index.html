URAI / LifeLogger System of Systems

Asset Factory production surface.

This deployment hosts the URAI asset generation control plane: request intake, durable metadata, queue records, Life Map event ingestion, Firestore and Storage rules, Firebase Functions, and live readiness checks.

Hosting shell live. Verify Functions at /api/health.

Asset Intake

POST /api/assets to create queued asset requests with metadata, tags, format, dimensions, and lifecycle state.

Asset Status

GET /api/assets/{assetId} to read current request status and storage manifest path.

Life Map Pipeline

POST /api/lifemap/events to ingest symbolic events for Life Map synthesis.

Operations

GET /api/health to verify functions, Firestore write access, version, and production readiness.
