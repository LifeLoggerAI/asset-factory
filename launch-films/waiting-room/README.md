# Waiting Room Launch Films

This directory contains the canonical production manifests for the fourteen-day URAI launch film system.

The production order begins with six anchor films:

1. Day 0 — The Doorway Exists
2. Day 1 — Movies of Life
3. Day 2 — This Is Real
4. Day 7 — Ask Me First
5. Day 8 — Life Becomes a Place
6. Day 14 — The World Opens

These films establish the recurring orb, doorway, memory, Passport, physical-world, and spatial-computing visual canon before the remaining daily films are generated.

No asset in this directory is public-release approved merely because it exists. Final publication requires the creative, product-truth, privacy, claims, accessibility, rights, spend, and publish gates defined in `docs/WAITING_ROOM_VIDEO_FACTORY_CONTRACT.md`.
